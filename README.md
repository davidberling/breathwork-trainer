# Breathwork Trainer

A progressive web app (PWA) for guided breathing exercises.

## Features

- **14 breathing techniques** across 5 categories (Calm & Sleep, Focus & Steady, Cooling & Reset, CO₂ Tolerance, Energize)
- **Visual breathing circle** that expands and contracts with each phase
- **Flip-clock number transitions** counting down each phase
- **Wood chime sounds** at phase transitions + soft clock ticks each second (optional, toggle on/off)
- **Wake Lock** keeps your screen on during sessions
- **Background timer** continues running even when the tab is backgrounded
- **Offline-capable** PWA — install to your home screen for a native app feel
- **Dark crimson theme** optimized for night use

## Usage

1. Visit the [live site](https://davidberling.github.io/breathwork-trainer/)
2. Select a breathing technique from the dropdown
3. Tap **Start** and breathe with the circle
4. Optionally enable sound with the speaker icon

### Install as an App

On your phone, open in Chrome (or Safari on iOS) and choose **Add to Home Screen** from the menu.
